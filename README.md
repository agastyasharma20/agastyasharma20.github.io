## 🔄 How to Update the Website

### Option A — Update via GitHub Website (Easy)
1. Go to your repo on github.com
2. Click on `index.html`
3. Click the ✏️ **pencil/edit icon** (top right)
4. Make your changes
5. Scroll down → click **"Commit changes"**
6. Site updates in ~1 minute ✅

### Option B — Upload a New File
1. Get the updated `index.html` file
2. Go to your repo → click **"Add file"** → **"Upload files"**
3. Upload the new file (it will replace the old one)
4. Click **"Commit changes"**
5. Site updates in ~1 minute ✅

## 📢 How to Add / Update Notices (No Coding Needed)

The website has a built-in **Notice Manager** — no code editing required!

1. Open the live website
2. Click the **"⚙️ Manage Notices"** button (bottom-left)
3. Add a new notice → choose type (Info / Warning / Alert)
4. Toggle existing notices on or off
5. Click **"✅ Save & Apply"** → ticker updates instantly

**Example notices to add:**
- `🎉 Clinic closed on 26th Jan for Republic Day`
- `⚠️ New evening timing from 1st April: 6:30 PM – 9:30 PM`
- `🩺 Free health camp on Sunday 20th April — Morning only`
